# LinkedIn

Use this folder for LinkedIn post drafts and published post history.

## What goes here

- `01-drafts/` - rough drafts and early versions
- `02-in-review/` - posts that are ready for cleanup or final edits
- `03-published/` - posts that are already live

## Voice

The assistant should use:
1. `VOICE.md`
2. `VOICE_linkedin.md` if it exists
3. relevant examples from `../_voice-examples/linkedin/` when helpful

## Keep it practical

Write clear posts, keep them specific, and avoid bloated filler.
