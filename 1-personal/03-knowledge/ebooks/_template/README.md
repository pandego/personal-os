# Ebook Folder Template

Use this template to keep each read ebook knowledge organized.

## Instructions
1. Duplicate this folder.
2. Rename the folder using the format: `[author-name]_[main-title]/`
3. Fill in the book details below.
4. Add the online resources if available, and your notes when you read the book.

## Structure
- `01-resources/` — original files (epub/pdf/audio), cover, metadata.
- `02-notes/` — your summaries, highlights, chapter notes.
- `_external/` — other files extracted or generated during reading (images, etc.)

## Notes conventions
- Name notes `chapter-<n>-<topic>-pXX-YY.md` (use page range if available).
