# Outbound

Use this folder for proactive business development and outreach work.

## What goes here

This is a broad workspace for outbound activity such as:
- outreach drafts
- campaign ideas
- follow-up sequences
- proposal notes
- reusable messaging patterns

## Keep it simple

Do not over-structure this area too early.
Start broad, then add subfolders only when repeated patterns make them useful.

## Working rule

If something becomes active client delivery work, move it to `2-business/03-clients/`.

## Quality bar

Before considering an outbound draft worth keeping, check:
- is the audience clear?
- is the ask small and easy to respond to?
- does it sound like a real person?
- could future-you reuse it without guessing context?
