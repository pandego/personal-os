# Voice Examples

This folder helps the assistant learn how you like to sound.

## What to put here

Add markdown files that are useful examples of your writing or examples worth learning from.

Use:
- `blog/` for blog-style examples
- `linkedin/` for LinkedIn-style examples

Examples can include:
- your own published writing
- strong drafts
- swipe examples
- rewritten versions you want the assistant to learn from

## Keep it simple

You do not need to label files as "best" or "swipe".
If an example is useful for voice, drop it in the right platform folder.

## How the assistant uses this

The assistant can read these examples when drafting content or when refreshing a platform voice guide.
