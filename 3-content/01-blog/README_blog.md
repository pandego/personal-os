# Blog

Use this folder for long-form articles and blog post drafts.

## What goes here

- `01-drafts/` - rough drafts and early versions
- `02-in-review/` - posts that are ready for cleanup or final edits
- `03-published/` - posts that are already live

## Voice

The assistant should use:
1. `VOICE.md`
2. `VOICE_blog.md` if it exists
3. relevant examples from `../_voice-examples/blog/` when helpful

## Keep it practical

Do not overthink the structure.
Write the draft, improve it, then move it forward when it is ready.
